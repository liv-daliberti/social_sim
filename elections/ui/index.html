Bayesian Network Sampling
Each node X has a set of parents Pa(X). We find the CPT row whose conditions are a subset of the current parent states — most specific match wins (highest count of matched conditions):
P(X = x | Pa(X) = pa)
Then draw: x ~ Categorical(P).
Root nodes A1, A2, A3 have no parents and use a fixed prior P(X) directly. All 23 nodes are sampled left-to-right in topological order so every parent is already resolved before its children.
Surprise (Self-Information)
After sampling x with probability p, how unexpected was that outcome?
S(x) = −log₂ p(x) [bits]
Low p → high surprise. Thresholds used here:
< 1 bit — expected outcome
1 – 2.5 bits — moderately surprising
⚡ > 2.5 bits — highly surprising
Edge Causal Influence (TVD)
For a group edge A → B, how much does A's sampled state shift B's probability distribution? For each child node b ∈ B that has a parent a ∈ A:
TVD(P, Q) = ½ · Σₓ |P(x) − Q(x)|
1. Group CPT rows by the value of parent a
2. Average distributions within each group
3. Compute TVD for every pair of groups
4. Average all pairwise TVDs across all (a, b) pairs
Range [0, 1]: 0 = parent irrelevant  ·  1 = parent fully determines child
Computed analytically from the CPT structure — not from simulation runs. The default visual weight is 50% (uniform prior on influence). Click any edge % to override it.
Hoeffding Validation
After n = 1 000 runs, Hoeffding's inequality bounds the chance that the empirical frequency deviates from the true probability by more than ε:
P(|P̂ − p| > ε) ≤ 2 · exp(−2nε²) Setting α = 1%: ε = √(−ln(α/2) / 2n) = √(−ln(0.005) / 2000) ≈ 5.15 percentage points
The two quantities being compared:
Model marginal = average of P(X | parents) over all n runs = E[P(X|Pa)]
Empirical freq. = observed count(X = x) / n
By the law of total expectation: E[P(X|Pa)] = P(X), so both quantities converge to the same value as n → ∞. A ✗ means the sampler's output disagrees with the CPT — a bug signal, not a model finding.
